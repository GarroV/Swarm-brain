# Подключение Claude Desktop к Swarm Brain

## Шаг 1 — Подключить MCP-сервер

Открой **Claude Desktop** → **Settings** → **Connectors** → **+**.

Заполни два поля:

- **Name**: `Swarm Brain`
- **Remote MCP server URL**: `https://vbqglndbxkpmreccpqmr.supabase.co/functions/v1/swarm-mcp`

Нажми **Add** → сервер появится в списке со статусом Connected и 14 инструментами.

---

## Шаг 2 — Создать проект с инструкциями

В боте напиши `/claude` — получишь готовый текст инструкций с уже подставленным твоим Telegram ID.

Затем в Claude Desktop: **Projects** → **New Project** → вставь этот текст в поле **Instructions**.

---

## Шаг 3 — Готово

Кидай транскрипт встречи или любой текст в проект → Claude покажет тезисы → проверь и скажи "сохрани" → данные уйдут в базу.

---

## Токен — опционально

По умолчанию сервер работает без токена: твоя личность определяется через `requesting_user_id` в аргументах инструментов (подставляется из инструкций проекта).

Если хочешь усилить безопасность (чтобы сервер сам тебя идентифицировал независимо от инструкций):

1. Напиши боту `/mytoken` — получишь токен вида `smcp_...`
2. В Claude Desktop → Connectors → Swarm Brain → **Advanced settings** → добавь заголовок:
   - Header: `Authorization`
   - Value: `Bearer smcp_...`

Со стороны сервера: токен принимается если передан, при совпадении — идентификация через него перекрывает `requesting_user_id` из аргументов.

## Доступные инструменты

| Инструмент | Что делает |
|---|---|
| `add_knowledge` | Добавить текст в базу (или личное хранилище) |
| `upload_file` | Загрузить файл в Storage + создать запись |
| `search_knowledge` | Найти информацию (включая личные записи) |
| `get_entry` | Полный текст записи по ID |
| `get_tasks` | Задачи команды с фильтрами |
| `get_meetings` | Последние встречи |
| `get_users` | Список команды с профилями |
| `get_storage_stats` | Статистика базы |
| `list_entries` | Список записей для ревизии |
| `delete_entry` | Удалить запись |
| `update_entry` | Обновить запись или заменить файл |
