# Подключение Claude Desktop к Swarm Brain

## Шаг 1 — Получить токен в боте

Открой бота в Telegram → напиши `/mytoken`.

Бот пришлёт твой персональный токен вида `smcp_xxxxxxxx-...`. **Скопируй его сразу** — повторно не показывается. Если потеряешь — запусти `/mytoken` снова (старый перестанет работать).

---

## Шаг 2 — Подключить MCP-сервер

Открой **Claude Desktop** → **Settings** → **Connectors** → **+** (добавить сервер).

Заполни поля:

| Поле | Значение |
|---|---|
| URL | `https://vbqglndbxkpmreccpqmr.supabase.co/functions/v1/swarm-mcp` |
| Header name | `Authorization` |
| Header value | `Bearer smcp_xxxxxxxx-...` (твой токен из шага 1) |

Нажми **Connect** — появится Swarm Brain со списком инструментов.

---

## Шаг 3 — Создать проект с инструкциями

В боте напиши `/claude` — получишь готовый текст инструкций с уже подставленным твоим Telegram ID.

Затем в Claude Desktop: **Projects** → **New Project** → вставь этот текст в поле **Instructions**.

---

## Шаг 4 — Готово

Кидай транскрипт встречи или любой текст в проект → Claude покажет тезисы → проверь и скажи "сохрани" → данные уйдут в базу.

---

## Доступные инструменты Swarm Brain

| Инструмент | Что делает |
|---|---|
| `add_knowledge` | Добавить текст в базу (или личное хранилище) |
| `upload_file` | Загрузить файл в Storage + создать запись |
| `search_knowledge` | Найти информацию (включая личные записи) |
| `get_entry` | Полный текст записи по ID |
| `get_tasks` | Задачи команды с фильтрами |
| `get_meetings` | Последние встречи |
| `get_users` | Список команды с профилями |
| `get_storage_stats` | Статистика базы |
| `list_entries` | Список записей для ревизии |
| `delete_entry` | Удалить запись |
| `update_entry` | Обновить запись или заменить файл |

## FAQ

**Почему бот говорит "токен сгенерирован" — и всё?**
Раньше была такая версия бота. Обновлено: теперь бот сразу показывает сам токен. Если старый токен был сгенерирован и не сохранён — запусти `/mytoken` снова.

**Нужно ли ещё раз подключать сервер если перевыпустил токен?**
Да. После `/mytoken` нужно обновить значение заголовка `Authorization` в настройках коннектора на новый токен.

**Зачем нужен токен, если есть Telegram ID?**
Токен — это доказательство что ты тот, за кого себя выдаёшь. Telegram ID передаётся в открытом виде, токен — нет. Если токен не совпадает, запрос отклоняется.
