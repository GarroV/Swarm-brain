# Swarm Brain

Командная база знаний с AI-поиском, управлением задачами и интеграцией митингов. Живёт в Telegram, работает на Supabase Edge Functions.

---

## Цель проекта

Собрать в одном месте всё, что знает команда — заметки, документы, договорённости, итоги встреч, ссылки — и сделать эту информацию мгновенно доступной через естественный язык. Бот работает там, где команда уже общается (Telegram), не требует переключения контекста.

---

## Функциональность

### База знаний
- **Добавление записей** — текст, ссылки, документы (TXT, MD, CSV, JSON, PDF), фото, голосовые сообщения
- **Семантический поиск** — поиск по смыслу через векторные эмбеддинги (OpenAI `text-embedding-3-small`)
- **AI-ответы** — бот отвечает на вопросы по содержимому базы (GPT-4o), умеет уточнять вопрос если контекста недостаточно
- **Автообработка медиа** — голос транскрибируется (Whisper), фото описывается (GPT-4o Vision), URL-адреса — содержимое страницы извлекается и сохраняется

### Задачи
- Просмотр задач с фильтрами: по исполнителю (`/tasks @Имя`), по тегу/стране (`/tasks Болгария`), по сроку (`/tasks week`)
- Смена статуса задачи через inline-кнопки (open → in_progress → done / cancelled)
- Удаление задач
- Задачи могут быть привязаны к митингу (создаются автоматически из Read.ai)

### Встречи (Read.ai)
- Автоматический вебхук: как только встреча завершается в Read.ai — её транскрипт и резюме сохраняются в базу знаний
- Просмотр последних встреч с кратким резюме
- Задачи из встречи отображаются рядом с резюме
- Удаление встречи вместе со всеми связанными задачами

### Пользователи и доступ
- Двухуровневая система: superadmin (владелец) и admin
- Управление доступом: `/users add`, `/users remove`, `/users list`
- Профили пользователей: имя, роль, рынки, телефон, email — редактируются через inline-кнопки

### MCP-сервер (Claude Desktop)
Экспонирует базу знаний как MCP-инструменты, доступные из Claude Desktop:
- `search_knowledge` — семантический поиск
- `get_tasks` — задачи с фильтрами
- `get_meetings` — последние встречи
- `get_users` — список команды с профилями
- `get_entry` — полный текст записи по ID
- `add_knowledge` — добавление записи

## Технический стек

| Слой | Технология |
|------|-----------|
| Runtime | Deno (Supabase Edge Functions) |
| База данных | Supabase (PostgreSQL + pgvector) |
| AI | OpenAI GPT-4o, Whisper, text-embedding-3-small |
| Интерфейс | Telegram Bot API |
| Митинги | Read.ai (OAuth2 + webhook) |

---

## Структура проекта

```
supabase/functions/
├── swarm-bot/        # Основной Telegram-бот
├── swarm-mcp/        # MCP-сервер для Claude Desktop
├── read-ai-auth/     # OAuth2 авторизация Read.ai
└── read-ai-webhook/  # Вебхук для приёма встреч из Read.ai
```
